# Voltio

## 📌 Descripción
Voltio es una aplicación enfocada en la gestión de entregas, productos y pedidos, permitiendo a los usuarios administrar operaciones de manera eficiente.

## 🛠️ Tecnologías
- Android (Kotlin)
- MQTT
- Backend (según configuración del proyecto)

## ⚙️ Requisitos
- Android Studio
- SDK actualizado

## 🚀 Instalación
1. Clonar el repositorio
2. Abrir en Android Studio
3. Ejecutar el proyecto

## 📂 Estructura
- `features/` → funcionalidades principales
- `core/` → lógica base
- `data/` → manejo de datos
